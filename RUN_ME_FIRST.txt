1. Extract this zip to a new folder.
2. Open a terminal in that folder.
3. Run:
   python3 -m venv .venv
   source .venv/bin/activate
   python3 -m pip install -r requirements.txt
   python3 -m streamlit run streamlit_app/app.py

   NOTE: requirements.txt now includes "-e ." which installs the agile_mc
   package from the src/ folder. This is required — without it the app will
   fail with "ModuleNotFoundError: No module named 'agile_mc'".

What changed:
- Sidebar now has a Project work % slider.
- How Many forecast text now shows Project and BAU counts on each confidence line.
- How Many probability chart now splits the bars into Project and BAU colours.
- When forecast uses the project percentage to reduce effective throughput.
